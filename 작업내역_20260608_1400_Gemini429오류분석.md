# 작업내역 - Gemini API 429 오류 분석

- **날짜/시간:** 2026-06-08 14:00
- **작업자:** Claude (AI)
- **프로젝트:** AI 멀티 모델 경쟁·토론 플랫폼

---

## 오류 내용

```
Gemini error: 429 You exceeded your current quota
Quota exceeded for metric: generativelanguage.googleapis.com/generate_content_free_tier_requests
limit: 0, model: gemini-2.0-flash
retry_delay: 35 seconds
```

---

## 진단 결과: 코드 문제 아님 — Google API 결제 문제 ❌

### 위반된 할당량 3가지

| 할당량 항목 | 할당량 ID | 상태 |
|------------|----------|------|
| 일일 요청 수 (무료 티어) | `GenerateRequestsPerDayPerProjectPerModel-FreeTier` | 초과 |
| 분당 요청 수 (무료 티어) | `GenerateRequestsPerMinutePerProjectPerModel-FreeTier` | 초과 |
| 분당 입력 토큰 수 (무료 티어) | `GenerateContentInputTokensPerModelPerMinute-FreeTier` | 초과 |

### `limit: 0` 의 의미

무료 티어 할당량이 완전히 소진되었거나, 해당 프로젝트/API 키에 대해 무료 티어 자체가 비활성화된 상태. 리셋을 기다려도 동일한 상황일 가능성 높음.

---

## 해결 방법

### ✅ 방법 1. Google Cloud 결제 활성화 (근본 해결)

1. [https://aistudio.google.com](https://aistudio.google.com) 접속
2. 우측 상단 → **Get API key** → 사용 중인 API 키 선택
3. **"Upgrade to pay-as-you-go"** 클릭
4. 결제 정보 등록 (신용카드)
5. 활성화 후 즉시 사용 가능

> Gemini 2.0 Flash 유료 요금: 입력 $0.075/1M tokens, 출력 $0.30/1M tokens (소액)

---

### ✅ 방법 2. 토론에서 Gemini 일시 제외 (즉시 가능, 무비용)

홈 화면 → 참가 AI 선택에서 **Gemini 체크 해제** → GPT-4o + Claude 조합으로 토론 진행.

> GPT-4o는 정상 동작 확인됨. Claude는 API 크레딧 충전 필요.

---

### ⏳ 방법 3. 일일 할당량 리셋 대기 (불확실)

무료 티어는 매일 자정 (태평양 시간) 리셋되나, `limit: 0` 상태는 리셋 후에도 동일할 가능성 높음. 권장하지 않음.
