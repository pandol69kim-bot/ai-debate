# 작업내역: Claude API 키 미등록 시 모델 카드 블러 처리

## 작업 일시
2026-06-08 16:00

## 작업 요약
초기 화면 AI 모델 선택 카드에서, Claude(Anthropic) API 키가 `.env`에 등록되지 않은 경우 해당 카드를 자동으로 블러 처리하도록 구현. 하드코딩 방식에서 백엔드 API 키 실제 등록 여부 기반의 동적 방식으로 전환.

---

## 변경 파일

### 1. `backend/app/schemas/schemas.py`
**변경**: `AIModelOut`에 `api_key_configured` 필드 추가

```python
class AIModelOut(BaseModel):
    id: UUID
    provider: str
    model_name: str
    display_name: str
    is_active: bool
    api_key_configured: bool = True  # ← 추가
```

---

### 2. `backend/app/api/routes/models.py`
**변경**: `GET /models/` 엔드포인트에서 provider별 API 키 등록 여부 계산 후 반환

```python
_API_KEY_MAP: dict[str, str] = {
    "gpt": settings.OPENAI_API_KEY,
    "claude": settings.ANTHROPIC_API_KEY,
    "gemini": settings.GOOGLE_API_KEY,
}

@router.get("/", response_model=list[AIModelOut])
async def list_models(db: AsyncSession = Depends(get_db)):
    result = await db.execute(select(AIModel).where(AIModel.is_active == True))
    models = result.scalars().all()
    return [
        AIModelOut(
            ...
            api_key_configured=bool(_API_KEY_MAP.get(m.provider, "")),
        )
        for m in models
    ]
```

- `bool("")` → `False` (키 없음)
- `bool("sk-ant-...")` → `True` (키 있음)

### 3. `frontend/src/types/index.ts`

```typescript
export interface AIModel {
  id: string;
  provider: string;
  model_name: string;
  display_name: string;
  is_active: boolean;
  api_key_configured: boolean;  // ← 추가
}
```

---

### 4. `frontend/src/app/page.tsx`
**변경 1**: 블러 조건을 하드코딩에서 동적으로 전환

```typescript
// 이전 (하드코딩)
const unavailable = model.provider === "claude";

// 이후 (동적)
const unavailable = !model.api_key_configured;
```

**변경 2**: 기본 선택 모델에서 API 키 미등록 모델 자동 제외

```typescript
// API 응답 기반
setSelectedModels(active.filter((m: AIModel) => m.api_key_configured).map((m: AIModel) => m.provider));

// 폴백 목록 기반
setSelectedModels(fallback.filter((m) => m.api_key_configured).map((m) => m.provider));
```

**변경 3**: 폴백 하드코딩 목록에 `api_key_configured` 필드 추가 (서버 미응답 시 대비)

```typescript
const fallback: AIModel[] = [
  { ..., provider: "gpt",    api_key_configured: true  },
  { ..., provider: "claude", api_key_configured: false },
  { ..., provider: "gemini", api_key_configured: true  },
];
```

## 적용 범위
- `.env`의 `ANTHROPIC_API_KEY` 값이 비어 있으면 Claude 블러 처리
- 키를 등록하고 **백엔드 서버 재시작** 시 자동으로 블러 해제
- GPT, Gemini도 동일한 로직 적용 (해당 키 없으면 동일하게 블러 처리됨)
