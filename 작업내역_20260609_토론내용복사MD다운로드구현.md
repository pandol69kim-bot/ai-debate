## 구현 상세

### MD 포맷 구조 (`debateToMarkdown.ts`)

```
# AI Arena 토론
**주제** / **날짜** / **참가 모델**

## Round 1
### 모델명
(내용)
> ⏱ 응답시간

## 심사 결과
승자 / 점수 표 / 심사 요약

## 합의 결론
신뢰도 / 최종 답변
```

### 복사 동작 (`useExportDebate.ts`)

- `navigator.clipboard.writeText` 우선 시도
- 미지원 환경은 `textarea + execCommand('copy')` fallback
- 복사 성공 시 2초간 `copied = true` 상태 유지

### 다운로드 동작 (`useExportDebate.ts`)

- `Blob` + `URL.createObjectURL` + `<a>` 클릭 방식
- 파일명: `토론_{주제 앞 40자}_{YYYY-MM-DD}.md`
- 파일명 특수문자(`/`, `?`, `:` 등) `_`로 치환 (`sanitizeFilename`)

### 버튼 UI (`ExportButtons.tsx`)

| 버튼 | 아이콘 | 상태 |
|------|--------|------|
| 복사 | `Copy` → 복사 성공 시 `Check` + "복사됨" (green) | 라운드 0개면 비활성화 |
| MD 저장 | `Download` | 라운드 0개면 비활성화 |

### 페이지 삽입 위치

토론 상세 페이지 헤더 우측 — 주제 제목과 같은 행에 배치

```tsx
<div className="flex items-start justify-between gap-4">
  <div className="flex-1">
    <h1>주제</h1>
    <StatusIndicator />
  </div>
  <ExportButtons topic={...} rounds={...} judge={...} consensus={...} />
</div>
```
