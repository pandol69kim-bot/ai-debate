- **날짜**: 2026-06-08
- **작업 시간**: 오후

---

## 작업 개요

상단 메뉴바(NavBar)에 모바일 반응형을 적용했습니다.
`md(768px)` 기준으로 데스크톱/모바일 레이아웃을 분기했습니다.

---

## 변경 파일

- `frontend/src/components/layout/NavBar.tsx`

---

## 구현 내용

### 데스크톱 (md 이상, 768px+)
기존과 동일한 가로 레이아웃 유지.

### 모바일 (md 미만)

**상단 바**
- 로고 (좌측)
- 로그인 시: 유저 이름 칩 표시
- 햄버거 버튼(☰) / 닫기 버튼(✕) 토글

**드롭다운 메뉴** (햄버거 클릭 시 펼침)
- 전체 nav 링크: Arena / 토론 기록 / Rankings / 관리자(관리자 계정만)
- 구분선
- 로그인 상태: 로그아웃 버튼
- 비로그인 상태: 로그인 / 회원가입 링크

**자동 닫힘 처리**
- `useEffect`로 `pathname` 변경 감지 → 메뉴 자동 닫힘
- 메뉴 항목 클릭 시 Link 이동과 함께 닫힘

---

## 주요 코드 변경

```tsx
// 상태 추가
const [menuOpen, setMenuOpen] = useState(false);

// 경로 변경 시 메뉴 닫기
useEffect(() => {
  setMenuOpen(false);
}, [pathname]);

// 데스크톱: hidden md:flex
// 모바일 햄버거: flex md:hidden
// 드롭다운: {menuOpen && <div className="md:hidden ...">}
```
