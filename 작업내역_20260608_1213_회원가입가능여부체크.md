## 응답 상세

### 회원가입 성공 (HTTP 200)
```json
{
  "id": "c70a1a67-cbd7-41c9-82c6-ca62e6e072da",
  "email": "korean2@arena.com",
  "name": "한글이름",
  "plan": "free",
  "created_at": "..."
}
```
- **한글 이름 UTF-8 저장 확인:** `name_len=4, utf8_bytes=12 → Korean OK = True` ✅
- 기본 플랜 `free` 자동 부여

### 중복 이메일 (HTTP 400)
```json
{"detail": "Email already registered"}
```

### 로그인 성공 (HTTP 200)
```json
{"access_token": "eyJhbGciOiJIUzI1NiIs..."}
```
- JWT 토큰 발급 정상
- 토큰 유효기간: 7일 (config.py `ACCESS_TOKEN_EXPIRE_MINUTES = 60*24*7`)

### 잘못된 비밀번호 (HTTP 401)
```json
{"detail": "Invalid credentials"}
```

## 주의 사항

### bcrypt 로그인 응답 지연
- bcrypt 해싱 연산으로 로그인 시 약 2~5초 소요 (정상 동작)
- httpx 기본 타임아웃(5초)이 부족할 수 있으므로 프론트엔드에서 **타임아웃 30초 이상** 권장

### PowerShell 콘솔 한글 표시 문제
- PowerShell 출력에서 한글이 `????`로 표시되는 현상은 **콘솔 인코딩 표시 문제**
- 실제 DB에는 UTF-8로 정상 저장됨 (octet_length 검증 완료)

## API 스키마

### 회원가입 요청
```json
POST /api/v1/auth/register
{
  "email": "user@example.com",   // EmailStr (필수)
  "name": "홍길동",               // string (필수)
  "password": "MyPassword123!"   // string (필수)
}
```

### 로그인 요청
```json
POST /api/v1/auth/login
{
  "email": "user@example.com",   // EmailStr (필수)
  "password": "MyPassword123!"   // string (필수)
}
```

### 로그인 응답
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs..."
}
```
> Authorization 헤더에 `Bearer {access_token}` 형식으로 사용
