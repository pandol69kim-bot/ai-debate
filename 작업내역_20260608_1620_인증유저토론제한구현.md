## 변경 파일

### 1. `backend/app/api/routes/debate.py`
**변경**: `POST /debate/start` 엔드포인트에 JWT 인증 의존성 추가

```python
# import 추가
from fastapi import Header
from typing import Optional
from app.api.routes.auth import _get_user_from_token

# 엔드포인트 시그니처 및 인증 체크
@router.post("/start", response_model=DebateStartResponse)
async def start_debate(
    body: DebateStartRequest,
    background_tasks: BackgroundTasks,
    db: AsyncSession = Depends(get_db),
    authorization: Optional[str] = Header(default=None),  # ← 추가
):
    await _get_user_from_token(authorization, db)  # ← 인증 실패 시 401
    ...
```

- Authorization 헤더 없음 → `401 Not authenticated`
- 토큰 만료 / 유효하지 않음 → `401 Invalid token`
- axios 인터셉터가 자동으로 `Bearer {token}` 첨부하므로 로그인 유저는 자동 통과

## 동작 시나리오

| 상황 | 동작 |
|------|------|
| 비로그인 → 화면 탐색 | 폼 표시 O, 안내 배너 표시, CTA → "로그인하고 토론 시작" |
| 비로그인 → CTA 클릭 | `/auth/login` 리다이렉트 |
| 비로그인 → API 직접 호출 | `401 Unauthorized` 반환 |
| 로그인 → 토론 시작 | 기존과 동일하게 정상 작동 |
| 로그인 → 토큰 만료 중 시도 | 에러 메시지 + `/auth/login` 리다이렉트 |
