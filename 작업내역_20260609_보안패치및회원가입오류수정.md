## 작업 2. 회원가입 후 자동 로그인 타입 오류 수정

### 문제

`frontend/src/app/auth/register/page.tsx` 49번줄

회원가입 성공 후 자동 로그인 시 `setAuth` 호출에 `is_admin` 필드 누락
→ TypeScript 빌드 오류 (`npm run build` 실패)

```
Type error: Argument of type '{ id: string; email: string; name: string; plan: string; }'
is not assignable to parameter of type 'AuthUser'.
Property 'is_admin' is missing in type ...
```

### 수정 내용

```typescript
// 수정 전
setAuth(loginRes.data.access_token, {
  id: user.id,
  email: user.email,
  name: user.name,
  plan: user.plan,
});

// 수정 후
setAuth(loginRes.data.access_token, {
  id: user.id,
  email: user.email,
  name: user.name,
  plan: user.plan,
  is_admin: false,   // 신규 가입자는 항상 일반 유저
});
```

### 원인

이전 작업에서 `AuthUser` 인터페이스에 `is_admin: boolean` 필드를 추가했으나
`login/page.tsx`만 수정하고 `register/page.tsx`는 누락됨
