- **프로젝트:** AI 멀티 모델 경쟁·토론 플랫폼

---

## 오류 내용

```
Gemini error: 404 This model models/gemini-2.0-flash is no longer available.
Please update your code to use a newer model for the latest features and improvements.
```

---

## 원인

`gemini-2.0-flash` 모델이 Google API에서 공식 폐기(deprecated & removed).  
이전 세션에서 `gemini-2.0-flash-exp` → `gemini-2.0-flash` 로 교체했으나, 이번에 `gemini-2.0-flash` 도 삭제됨.

**Gemini 모델 폐기 이력:**

| 단계 | 모델명 | 상태 |
|------|--------|------|
| 1차 | `gemini-2.0-flash-exp` | ❌ 삭제됨 (이전 세션 수정) |
| 2차 | `gemini-2.0-flash` | ❌ 삭제됨 (이번 세션 수정) |
| 현재 | `gemini-2.5-flash` | ✅ 적용 |

---

## 수정 내용

### 수정된 파일 4곳

| 파일 | 변경 전 | 변경 후 |
|------|--------|--------|
| `.env` | `GEMINI_MODEL=gemini-2.0-flash` | `GEMINI_MODEL=gemini-2.5-flash` |
| `backend/app/core/config.py` | `GEMINI_MODEL: str = "gemini-2.0-flash"` | `GEMINI_MODEL: str = "gemini-2.5-flash"` |
| `backend/app/adapters/gemini_adapter.py` | `display_name = "Gemini 2.0"` | `display_name = "Gemini 2.5"` |
| DB `ai_models` 테이블 | `model_name='gemini-2.0-flash'` | `model_name='gemini-2.5-flash'` |

### DB 업데이트 확인

```
('claude', 'claude-opus-4-8', 'Claude Opus')
('gemini', 'gemini-2.5-flash', 'Gemini 2.5')   ← 수정됨
('gpt', 'gpt-4o', 'GPT-4o')
Done.
```

## 주의 사항

모델명 404 오류는 수정됐으나, **Gemini 무료 티어 할당량 초과(429)** 문제는 별개.  
실제 Gemini 응답을 받으려면 Google AI Studio에서 결제 활성화 필요.

| 오류 코드 | 원인 | 해결책 |
|----------|------|--------|
| 404 | 모델명 폐기 | ✅ 이번 수정으로 해결 |
| 429 | 무료 티어 할당량 초과 | Google 결제 활성화 필요 |
