## 발견된 버그 및 수정 사항

### Bug 1. bcrypt 버전 충돌 → 회원가입 500 오류

**증상:** `POST /api/v1/auth/register` → HTTP 500

**원인:**
```
ValueError: password cannot be longer than 72 bytes
```
`passlib==1.7.4` + `bcrypt==5.0.0` 호환 불가.  
bcrypt 4.x부터 API 변경으로 passlib의 내부 wrap bug 감지 로직에서 충돌.

**수정:**
```powershell
pip install bcrypt==4.0.1   # 5.0.0 → 4.0.1
```
`requirements.txt`에 `bcrypt==4.0.1` 명시 추가.

---

### Bug 2. DebateStartRequest 스키마 오해

**증상:** 토론 시작 시 `model_ids`(UUID 배열)로 요청 → HTTP 422

**실제 스키마:**
```json
{
  "topic": "string",
  "selected_models": ["gpt", "claude", "gemini"],  // provider 이름 문자열
  "num_rounds": 3
}
```
→ `selected_models`는 UUID가 아닌 **provider 이름** 문자열 배열

---

### Bug 3. Gemini 모델명 deprecated

**증상:** `GeminiAdapter.generate()` → HTTP 404

**원인:** `gemini-2.0-flash-exp` 모델이 API에서 제거됨

**수정:**
- `.env`: `GEMINI_MODEL=gemini-2.0-flash-exp` → `gemini-2.0-flash`
- `config.py`: 동일 수정
- DB `ai_models` 테이블: `UPDATE` 쿼리로 model_name 업데이트
